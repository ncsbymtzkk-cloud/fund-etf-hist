# ETF 历史行情查询

> 基于 AKShare `fund_etf_hist_sina` 接口封装

## 功能

获取指定ETF基金的日频率历史行情数据（开盘价、最高价、最低价、收盘价、成交量）。

## 输入参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| symbol | str | ETF基金代码 | `"sh510050"`（上证50ETF）、`"sz159915"`（创业板ETF） |

### 常见ETF代码

| 基金代码 | 名称 |
|---------|------|
| sh510050 | 上证50ETF |
| sh510300 | 沪深300ETF |
| sh510500 | 中证500ETF |
| sh588000 | 科创50ETF |
| sz159915 | 创业板ETF |
| sz159845 | 中证1000ETF |
| sh518880 | 黄金ETF |
| sh511880 | 货币ETF |

## 输出参数

| 列名 | 类型 | 说明 |
|------|------|------|
| date | object | 日期 |
| open | float64 | 开盘价 |
| high | float64 | 最高价 |
| low | float64 | 最低价 |
| close | float64 | 收盘价 |
| volume | int64 | 成交量（单位：手） |

## 使用方式

问AI："查一下上证50ETF的历史数据" 或 "获取510050的行情数据" 即可自动调用。

## 注意

- 单次返回该基金所有历史数据（自上市以来）
- 数据来源于新浪财经
- 成交量单位为"手"（1手 = 100份）
